COMO IMPORTAR PARA O RENDER POSTGRESQL

1. Copie o Internal Database URL do banco no Render.
2. No PowerShell, dentro desta pasta, rode:

   $env:DATABASE_URL="COLE_AQUI_O_INTERNAL_DATABASE_URL"

3. Instale a dependência, se necessário:

   python -m pip install psycopg2-binary

4. Rode a importação:

   python importar_json_para_postgres.py

https://ponto.petegato.com.br/login?chave=SUA_CHAVE

IMPORTANTE:
- Não use a URL do site em DATABASE_URL.
- Use somente o Internal Database URL do PostgreSQL.
- O script atualiza registros já existentes e não duplica pelo ID.
