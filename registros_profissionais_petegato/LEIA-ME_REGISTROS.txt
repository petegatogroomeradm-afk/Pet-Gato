REGISTROS PROFISSIONAIS - PET & GATÔ

O que este pacote faz:
- Substitui a tela de Registros por uma versão profissional.
- Agrupa os registros por funcionário e dia.
- Permite ajustar a jornada completa: entrada, saída intervalo, retorno intervalo e saída.
- Valida horários inconsistentes antes de salvar.
- Mantém auditoria na tabela time_adjustments.
- Mantém o ajuste individual antigo no histórico bruto.

COMO APLICAR:
1. Extraia este ZIP.
2. No PowerShell:

cd "D:\Relogio de ponto WEB"
python "CAMINHO_DA_PASTA_EXTRAIDA\aplicar_registros_profissionais.py"

git add .
git commit -m "Adiciona registros profissionais"
git push

Depois aguarde o Render atualizar.
